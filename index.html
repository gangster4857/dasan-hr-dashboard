⚙️ Tally → Google Sheets → 대시보드 연동 설정
노션 없이, Tally → 구글 스프레드시트 직접 연동. 대시보드가 시트 데이터를 실시간으로 읽어옵니다.
1
Tally 폼 → Google Sheets 자동 연동
Tally 폼 편집 화면 → Integrations → Google Sheets 선택
→ Google 계정 연결 → 새 스프레드시트 생성 또는 기존 시트 선택
→ 시트 이름을 채용 DB로 설정
→ Save 클릭

이후 지원자가 Tally 폼 제출 시 → 시트에 자동으로 행이 추가됩니다.
전형단계 관리: 시트 A열에 전형단계 컬럼을 추가하고, 데이터 → 데이터 유효성 검사 → 드롭다운으로 서류검토중, 서류합격, 면접진행, 최종합격, 불합격, 보류 설정.
💡 파일(이력서 등)은 Tally 서버에 저장되고 시트에는 다운로드 링크 URL이 저장됩니다. 링크를 클릭하면 파일을 바로 열 수 있습니다.
2
스프레드시트를 "링크 있는 사용자 모두 보기"로 공유
스프레드시트 우측 상단 공유 → 링크 있는 사용자 → 뷰어로 설정
→ 링크 복사

대시보드는 이 공개 링크를 통해 데이터를 읽습니다. 보기 전용이므로 외부에서 수정할 수 없습니다.
✅ 전형단계 변경은 인사팀이 구글 시트에서 직접 드롭다운으로 선택하면 대시보드에 자동 반영됩니다.
3
스프레드시트 ID 입력 → 대시보드 연동
시트 URL에서 ID를 확인하세요:
https://docs.google.com/spreadsheets/d/여기가 시트 ID/edit
시트 이름(탭 이름)도 정확히 입력하세요.
스프레드시트 ID
시트 이름 (탭)
컬럼 매핑 — 시트 헤더명과 정확히 일치해야 합니다
현재 시트 컬럼 구조(A~S)에 맞게 기본값이 설정되어 있습니다.

4
자동 갱신 설정
대시보드는 기본적으로 60초마다 자동 새로고침합니다.
구글 시트에서 전형단계를 변경하면 1분 내에 대시보드에 반영됩니다.
즉시 반영하려면 상단 🔄 새로고침 버튼을 클릭하세요.
⚠️ Google Sheets JSON API는 공개 공유된 시트에서만 동작합니다. 시트 공유 설정이 "링크 있는 사용자 - 뷰어"인지 확인하세요.
