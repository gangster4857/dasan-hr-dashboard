⚙️ Tally → Notion → 대시보드 연동 가이드
아래 순서대로 설정하면 지원자가 Tally 폼 제출 시 Notion DB에 자동 저장되고, 이 대시보드에서 관리할 수 있습니다.
1
Notion DB 생성
노션 채용허브 페이지 안에 + 새 데이터베이스를 만드세요.
아래 컬럼(속성)을 추가합니다:
Notion 속성명타입Tally 연동 필드명
이름제목(Title)이름
공고명선택(Select)지원_공고
성별선택(Select)성별
생년월일날짜(Date)생년월일
연락처전화번호(Phone)연락처
이메일이메일(Email)이메일
학력선택(Select)학력
전공텍스트(Text)전공
경력선택(Select)경력기간
직전직장텍스트(Text)직전_직장명
직전연봉숫자(Number)직전_연봉_만원
희망연봉숫자(Number)희망_연봉_만원
전형단계선택(Select)— (수동 관리)
메모텍스트(Text)— (수동 관리)
지원일시날짜(Date)자동 생성
2
Tally 폼 생성 (무료)
tally.so에서 회원가입 후 새 폼 생성.
위 Notion DB 컬럼과 동일한 필드명으로 질문을 추가합니다.
파일 업로드 필드: 이력서 / 자기소개서 / 포트폴리오 (각각 별도 질문)
공고명 필드: 선택지에 공고명을 제형개발연구원 형태로 추가 (공고 추가 시 선택지 추가)
💡 공고를 여러 개 운영할 때는 폼 하나에 "지원 공고 선택" 질문을 추가하거나, 공고마다 별도 Tally 폼을 만들어도 됩니다. 둘 다 같은 Notion DB로 연동 가능합니다.
3
Tally → Notion 자동 연동 설정
Tally 폼 편집 화면 → Integrations → Notion 선택
→ Notion 계정 연결 → 위에서 만든 DB 선택 → 필드 매핑
→ Save 클릭

이후 폼 제출 시 자동으로 Notion DB에 행이 추가됩니다.
⚠️ 파일(이력서 등)은 Tally 서버에 저장되고 Notion에는 파일 링크(URL)가 저장됩니다. Notion에서 링크 클릭 시 파일 다운로드 가능합니다.
4
Notion API 키 발급 (대시보드 연동용)
notion.so/my-integrations → 새 통합 만들기
→ 이름: 채용 대시보드 → 생성
→ Internal Integration Token 복사
→ 노션 채용 DB 페이지 → 우측 상단 ··· → 연결 추가 → 통합 선택
→ 아래 설정 칸에 토큰과 DB ID를 입력하세요.
5
대시보드에 API 키 입력
아래에 입력하면 이 대시보드가 Notion DB 데이터를 실시간으로 읽어옵니다.
Notion Integration Token
Notion Database ID
⚠️ 현재 대시보드는 브라우저 로컬 스토리지(Artifact Storage)를 사용합니다.
Notion API는 CORS 제한으로 브라우저에서 직접 호출이 어렵습니다.
실제 운영 권장 방법: GitHub Pages + Cloudflare Workers(무료)로 프록시 설정하거나, Make(Integromat) / Zapier를 이용해 Notion → Google Sheets 자동 동기화 후 Google Sheets API로 연결하면 됩니다. 필요 시 가이드를 요청해주세요.
